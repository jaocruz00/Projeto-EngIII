Esse diretório é refere-se ao Projeto final da materia de ENGIII no curso de Analise e Desenvolvimento de Sistema da FATEC Ourinhos

Nessa faze no projeto estamos pondo em prática a parte técnica elaborado nos semestre anteriores, seguindo a documentação elaborada na matéria de Eng de Software II, com a fiscalização dos docentes das matérias. 

A descrição, as especificações e os modelos referentes ao desenvolvimento de um sistema de gerenciamento e controle de frequência e eventos, voltado para atividades de grupos, como corridas de rua, escoteiros, eventos de credenciamento em gera. nesse modelo, usamos como base um grupo de jovens pertencente a uma comunidade religiosa.
O sistema proposto tem como objetivo auxiliar os coordenadores na organização e no acompanhamento da participação dos membros em encontros e demais atividades promovidas. Por meio da plataforma, será possível cadastrar usuários, registrar presenças, consultar informações de frequência e acessar dados individuais dos participantes, contribuindo para uma gestão mais eficiente e centralizada.

Cabe ressaltar que até o presente modelo (3.0) não há a implementação de Banco de Dados, pois o objetivo do projeto é ilustrar as atividades necessarias descritas na documentação, por este motivo, ultilizamos scrips tem javaScrip para representar as ações executadas e posteriormente em novas versões, cadastradas e validadas no banco de dados (como o cadastro de participantes, validação, validação de pagamento da inscrição,etc).
